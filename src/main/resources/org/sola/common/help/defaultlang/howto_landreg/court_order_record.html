Record Court Order

The Record Court Order service allows you to register a court order over a property

Steps

    1.  Start the Record Court Order  service. The Property Details screen will open.
    2. On the Rights / Restrictions tab the Court Order right will be selected as the default Right type. If not, select it from the dropdown list. Click theCreate tool to create a pending Court Order right and open the Court Order screen.
    3. In the Court Order screen fill in the Registration date, add a Court Order document and update the Memorial text.
    4. ClickCreate & Save and close the Court Order screen.
    5. If you need to make further changes to the court order information, select the pending Court Order in the Rights / Restrictions tab and click theEdit tool.
    6. If you need to remove the pending Court Order details completely, use theRemove tool on the Rights / Restrictions tab.
    7. Save changes to Property Details andComplete the service.
