Change Court Order

If you need to change the details of a registered Court Order, use the Record Right or Restriction service. This service allows you to change the details for any right or restriction.

Steps

    1.  Start the Change Right or Restriction service. The Property Details screen will open.
    2. On the Rights / Restrictions tab, select the Court Order right you intend to change. If the Court Order right is current, theVary tool will enable. ClickVary to open the Court Order screen.
    3. Update the Court Order details as appropriate and set the Memorial text. If the court order was migrated from LRS, you may need to add document details before you can save your changes.
    4. ClickCreate & Save and close the Court Order screen.
    5. Varying a right creates a new pending version of that right. If you need to make further changes to the court order information, select the pending Court Order in the Rights / Restrictions tab and click theEdit tool.
    6. If you need to remove the pending Court Order details completely, use theRemove tool on the Rights / Restrictions tab.
    7. Save changes to Property Details andComplete the service.
