Change Common Property

There are several situations that can trigger a change to the extent of the Common Property. These include

    • Adding land to or removing land from the Common Property
    • Dissolving one or more units into the Common Property
    • Subdividing the Common Property to create new units

The Body Corporate may also sell or grant a lease over the Common Property and/or grant an easement.

Add or Remove Land from the Common Property

To add or remove land from the Common Property will first require subdivision and/or amalgamation of the parcel from the underlying property (i.e. the parcel underlying the Unit Title Development), which in turn requires cancellation of the underlying property title and the creation of the new property titles. These actions must occur before it is possible to update the Common Property details.

Important: Never cancel the Common Property folio using the Cancel Title service. The Common Property is required by SOLA to link the unit title folios to the underlying property as well as record Body Corporate information (i.e. Rules and Address for Service). If all land is taken from the Common Property set the official area on the Common Property folio to 0.

The following describes the general steps that need to be completed. Detailed information for each general step can be found in the appropriate How To section.

General Steps

    1. Create or request a survey plan that shows how the underlying parcel is being subdivided and/or amalgamated and use the Record Plan service to capture the details of this plan. This will result in the underlying parcel of the Unit Title Development becoming historic/cancelled when the survey plan is approved.
    2. Create new titles for the subdivided/amalgamated land using the Create New Titles service and use the Cancel Title service to cancel the folio of the property underlying the Unit Title Development.
    3. Lodge a Redevelopment Plan that shows the new area for the Common Property using a Record Unit Plan service.
      1. On the Unit Titles screen, go to the Map tab, de-select the original underlying parcel and select the recently created subdivided/amalgamated parcel as the new underlying parcel for the Unit Title Development.
      2. Update the Official Area of the Common Property parcel.
      3. If there has been any units added to, or removed from the development,Add the necessary unit parcels and/orRemove existing parcels as required.
    4. Lodge an application with the Create Unit Titles service and edit the Common Property folio.
      1. Update the area for the Common Property folio to reflect the new official area of the Common Property parcel.
      2. Use the tools on the Related properties tab to disassociate the original Prior title reference and add a reference to the new underlying property as the Prior title for the Unit Title Development. 
      3. Update the Corporate Body Rules as required.
      4. Update the Unit Entitlements on the Principal Unit folios as required.
      5. If new units have been created by the redevelopment, use theCreate tool to create new stub folios and edit the folios to add Unit Entitlements and any other relevant details.
      6. If necessary, use the Cancel Title service to cancel any Principal Unit folios that may have been removed from the Unit Title Development.

Dissolving Units and/or Subdividing the Common Property

For SOLA, dissolving units into the Common Property and/or further subdividing the existing Common Property to create new units is the same as processing a Redevelopment Plan. Refer to the Lodge a Redevelopment Plan How To section for details on how to deal with this situation.

Leases and Easements over Common Property

SOLA treats the Common Property folio the same as any other unit or property folio. You can register any transaction to the Common Property folio including Leases and Easements by using the appropriate service (i.e. Record Lease, Record Easement). Refer to the Land Registration How To for further information.
