Cancel Transmission

To cancel a Transmission right, use the Cancel Transmission service or the Cancel Transmission under Transfer by Administrator service.

Steps

    1. Start the service used to cancel the transmission. The Property Details screen will open.
    2. On the Rights / Restrictions tab, select the Transmission right you intend to cancel. If the Transmission right is current and there are no pending changes for the Transmission , theExtinguish tool will enable. Click Extinguish to open the Transmission screen. 
    3. Update the transmission details as appropriate.
    4. ClickCreate & Save and close the Transmission screen.
    5. Canceling a right creates a new pending version of that right. If you need to make further changes to the transmission information, select the pending Transmission in the Rights / Restrictions tab and click theEdit tool.
    6. If you need to reverse the cancellation, use theRemove tool on the Rights / Restrictions tab or remove the pending right.
    7. Save changes to Property Details andComplete the service
