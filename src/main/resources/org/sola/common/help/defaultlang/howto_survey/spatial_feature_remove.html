Removing a Road, Village or Hydro Parcel from the Map

From time to type it may become necessary to remove a road, village or hydro parcel from the map. You can do this using either the Record Plan service or the Change Map service. Simply select the spatial feature to remove using theSelect Spatial Feature tool and check the Delete on approval check box in the Edit Spatial Features dialog. As with changing the name, the spatial feature will remain displayed on the map until the application is approved.
