Correct the Registry

The Correct Registry services can be used to correct erroneous details on the property on behalf of the Registrar. They allow

    • any rights or restrictions to be added, changed or cancelled
    • the estate type of the property to be changed
    • memorials to be added to the property
    • the prior title and village parent property details added, changed or removed
    • the property area to be modified

These services do not allow the property to be cancelled or the folio reference to be changed. To cancel a property, use the Cancel Title service. Changing the folio reference is not recommended as there may be legal documents that reference the folio. Instead the property should be cancelled and a new property with a different folio created in its place (using the Create New Title service).

SOLA does not allow the same service to create, change and cancel rights. For this reason there are two Correct Registry services, Correct Registry and Correct Registry (Cancel Right). These services can be used independently or in combination as required.

An Office Note must accompany these services to describe and justify the changes made to the property. No fee is charged for these services.

Use the Correct Registry service to record new rights, change existing rights, add property memorials, change prior title and village parent property details and edit the property area.

Steps

    1. Start the Correct Registry service. The Property Details screen will open.
    2. Use the Rights / Restrictions tab,Create orVary rights and restrictions as appropriate. 
    3. Add a new memorial to the property from the Memorials tab if required.
    4. Update the Village and/or Prior Title from the Property history tab if required. Note that changes to the Parent Properties will be reflected on the live property record immediately upon saving the Property Details screen.
    5. On the General tab, update the property area if required. Note that changes to the property area will be reflected on the live property record immediately upon saving the Property Details screen.   
    6. Save changes to Property Details andComplete the service. Changes to Parent Properties or the property area will be reflected on the live property record immediately. Changes to the rights and restrictions or the property memorials will not be reflected until the application is approved.
