Getting help with SOLA Desktop

SOLA includes a comprehensive help file. If you need to refer to any additional information regarding SOLA functionality, use Context Help under the Help menu. Useful help topics include the Public User How To as well the SOLA Desktop Screens > SOLA Map Viewer and SOLA Desktop Screens > Document Search.

If you encounter an issue with SOLA that you are not able to resolve, report the issue to the Public Access Computer Administrator for assistance
