Change Easement

If you need to change the details of a registered easement, use the Change Right or Restriction service. This service allows you to change the details for any right or restriction.

Steps

    1.  Start the Change Right or Restriction service. The Property Details screen will open.
    2. On the Rights / Restrictions tab, select the Easement right you intend to change. If the Easement right is current, theVary tool will enable. ClickVary to open the Easement screen.
    3. Update the easement details as appropriate and set the Memorial text. If the easement was migrated from LRS, you may need to add document details before you can save your changes.
    4. ClickCreate & Save and close the Easement screen.
    5. Varying a right creates a new pending version of that right. If you need to make further changes to the easement information, select the pending Easement in the Rights / Restrictions tab and click theEdit tool.
    6. If you need to remove the pending Easement details completely, use theRemove tool on the Rights / Restrictions tab.
    7. Save changes to Property Details andComplete the service.
