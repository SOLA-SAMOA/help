Linking property

Property Details

You can create a new property out of an existing property using either the New Freehold title, New Apartment title, Register New Ownership or New State service. Using an assigned application, start the new property related service from the Application Details Services tab to access the linking property functionality.


You are asked whether you want to link prior property objects to the new one

  • a) If you answer yes and there is no previous task on the property already running the Property screen for selecting the prior property/title to be linked to new property from an Application's Properties list tab or from a Search tab list is opened;
  • b) else the Property Details screen is opened with information in top about: Application number, Property, Service and Property details and all the tabs for input relevant data

After you have selected the prior property you want to supersede, you can select the rights and the parcels to be linked to the new property also specifying the type of relationships between instances of the property (e.g. Prior Title, Root Title.)


A message reminding to make changes in the current property object will be opened. in order to do this, once the new property change status from pending to current, click Terminate in the Property Form of the parent property.
