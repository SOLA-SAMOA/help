Print a Unit Title Certificate

You can print a Unit Title Development Certificate from the Unit Titles screen.

Steps

    1. Use Property Search to locate one of the properties for the Unit Title Development such as the Common Property and open Property Details.
    2. The Open Units tool should appear beside thePrint Options dropdown in Property Details. Click theOpen Units tool to open the Unit Titles screen. 
    3. Click thePrint tool in the Unit Titles screen to generate the Unit Title Development Certificate.

Important: The Unit Title Development Certificate currently available from the Unit Titles screen is a draft/training certificate only. MNRE will need to review and advise on the appropriate content and layout of this certificate before it can be used as an official MNRE product.
