Change Miscellaneous

If you need to change the details of a registered miscellaneous right, use the Change Right or Restriction service. This service allows you to change the details for any right or restriction.

Steps

    1.  Start the Change Right or Restriction service. The Property Details screen will open.
    2. On the Rights / Restrictions tab, select the Miscellaneous right you intend to change. If the right is current, theVary tool will enable. ClickVary to open the Miscellaneous screen.
    3. Update the memorial details as appropriate.
    4. ClickCreate & Save and close the Miscellaneous screen.
    5. Varying a right creates a new pending version of that right. If you need to make further changes to the memorial, select the pending Miscellaneous right in the Rights / Restrictions tab and click theEdit tool.
    6. If you need to remove the pending Miscellaneous right completely, use theRemove tool on the Rights / Restrictions tab.
    7. Save changes to Property Details andComplete the service.
