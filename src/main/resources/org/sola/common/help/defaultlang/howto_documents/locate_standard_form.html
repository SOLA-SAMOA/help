Locate a Standard Form

SOLA includes copies of all MNRE Standard Forms. To locate one use the Document Search. 

Steps

    1. From the SOLA Main Menu, clickDocument search to open the Document Search screen.
    2. Set the Type field to Standard Form and optionally enter the name of the form in the Description field. Click Search.
    3. Select the desired form from the search results list and clickOpen attachment
    4. The form will display in the default document viewer. Print or save the form as required.
