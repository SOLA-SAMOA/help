Change Caveat

If you need to change the details of a registered caveat, use the Change Caveat service. This service must be the only service on the application otherwise the SOLA business rules will prevent successful completion of the service (i.e. step 7 will fail).  

Steps

    1.  Start the Change Caveat service. The Property Details screen will open.
    2. On the Rights / Restrictions tab, select the Caveat restriction you intend to change. If the Caveat restriction is current, theVary tool will enable. ClickVary to open the Caveat screen. 
    3. Update the caveat details as appropriate and set the Memorial text. If the caveat was migrated from LRS, you may need to populate the right holder and document details before you can save your changes.
    4. ClickCreate & Save and close the Caveat screen.
    5. Varying a right creates a new pending version of that restriction. If you need to make further changes to the caveat information, select the pending Caveat in the Rights / Restrictions tab and click theEdit tool.
    6. If you need to remove the pending Caveat details completely, use theRemove tool on the Rights / Restrictions tab.
    7. Save changes to Property Details andComplete the service.
