Remove a Parcel from the Map

The Change Parcel Attributes tool can also be used to remove a parcel from the map by making it historic. This action may be required if the parcel was mistakenly omitted as a Target Parcel when recording a new plan.

WARNING Care should be taken when removing a parcel from the map. Once the parcel is historic, no further changes to the parcel will be possible. If you make a mistake and select the wrong parcel, it will result in a hole in the map that can only be fixed via a database script.

To remove a parcel, follow the steps noted above for changing the Lot or Plan number for a parcel using theChange Parcel Attributes tool and clickMake Historic. You will be prompted to confirm your choice. Once confirmed, the parcel will be made historic. Refresh the map and you should see that the parcel no longer appears.
