Release 2001a

This section summarises the changes made to the SOLA Samoa Desktop application or it’s configuration for Release 2001a.

Ref#

Title

Summary

A

Add Samoa Aerial Photos from 2015

Aerial Photos from 2015 for all of Samoa have been added to the SOLA Map Viewer using Geoserver. Users must have the View Aerial Photos security role to see the new map layer. 

B

Add Geodetic Marks to the Map

Geodetic Marks for Samoa have been added to the SOLA Map Viewer. Users can also search for Geodetic Marks from the Map Find and view the Geodetic Mark coordinates in the Information Tool.  

C

Capture Traverse Marks

New map editing tools have been added to allow capture of survey traverse marks. The Information Tool will also display the coordinates for traverse marks.

D

Capture District Boundaries

New map editing tools have been added to allow capture of district boundaries.

E

Certificate of Title

A new Certificate of Title (CoT) report has been added along with a stored procedure (administrative.show_cot_report) to control when the CoT report becomes active.

F

Public User Profile

Several changes made to support a public user profile for SOLA. Includes restricted access to specific document types.

G

Public User Activity Report

A new report added to display the activity of public users. Intended to support fee reconciliation and general queries around the number of active public users.

H

Report Improvements

General improvements to reporting functionality including support for multiple fonts to ensure printed reports appear the same as when displayed onscreen.

I

Make Historic Deeds Current

Team Leaders with the Make Property Current security privilege can make historic deeds current to allow transactions to be processed on them.

J

Dockerise SOLA

SOLA infrastructure has been upgraded to run on Docker containers so that future component upgrades will be much easier.
