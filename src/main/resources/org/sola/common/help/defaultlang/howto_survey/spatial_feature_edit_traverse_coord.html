Edit a Traverse Mark Coordinate

The coordinate of a traverse mark can be edited using the Spatial Feature tools. You can do this using the Record Plan or Change Map service. The steps below illustrate this using the Change Map service.

Steps

    1. Start the Change Map service to open the Change Map form. Use the Map Viewer to locate the the traverse mark to update and select it using using theSelect Spatial Feature tool. This will highlight the feature with a dark blue boarder and light yellow fill.
    2. To edit the traverse mark coordinate, use theEdit Spatial Feature Details tool to update either or both of the Northing, Easting values.
    3. The new position for the traverse mark will not display on the map until the application is approved so the final steps are to complete the service and approve the application.
