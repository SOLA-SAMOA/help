Adding or Changing a Road, Village or Hydro Parcel

Several map tools are provided as part of Record Plan and Change Map that allow you to add and edit roads, villages and hydro parcels. To add any of these features to the map, select the appropriate map tool;Add New Hydro Parcel tool,Add New Road Parcel tool,Add New Road Centerline tool orAdd New Village tool and use the mouse to create the feature.

If you need to change or move one of these features you can useEdit Spatial Feature tool.  First select the spatial feature to edit using theSelect Spatial Feature tool. Once the feature is selected, its shape can be changed using theEdit Spatial Feature Tool by dragging any of the features node points (i.e. light red squares) with the mouse. To add a new node point, hold the CTRL key and click one of the boundary lines of the feature. To remove a node point. Hold the SHIFT key and click the node to remove. Using theEdit Spatial Feature Tool allows you to reshape the feature as you see fit.

If you make a mistake and need to clear the changes you have made, either deselect  the edited feature using theSelect Spatial Feature tool or use theEdit Spatial Feature Details tool toRemove the edited feature from the service.

Be aware that although you can see the changes to the shape of the feature, they will not be reflected on the map for others to see until the application is approved.
