Release 1405a

This section summarises the changes made to the SOLA Samoa Desktop application or it’s configuration for Release 1405a. Further details for each change can be found in the SOLA Samoa Lighthouse Project (https://flossola.lighthouseapp.com/projects/92838-samoa/overview) using the Lighthouse Ticket # as reference.

Ticket #

Title

Change Summary

67

Create Folio Certificate for Unit Titles Development

Draft Unit Title Development folio certificate created. Further customisation may be required when the first Unit Title Development is submitted. 

68

Unit Title – Rights on the Freehold Parcel should be Cancelled

Significant improvements to Unit Title functionality including support for Unit Titles cancellation where the Unit Title ownership structure is transferred to the underlying freehold parcel at the point of cancellation.

69

Parcel editing

Added a new map tool (Change Parcel Attributes) to allow users with the Change Parcel Attribute Tool role to edit the lot, plan number and area for a parcel. 

87 Update SOLA Samoa Help documentation Added help for Unit Titles as well as other new features the SOLA Samoa User Guide. Updated the online help file for the SOLA Desktop to reflect the latest User Guide. Added full text search feature to the online help file.

113

Tool to make parcels historic in Change Map

New map tool (Change Parcel Attributes) also supports making current parcels historic.

137

Lodgment Report Updates

Several change to the Lodgment Report including;

  1. Displaying the number of applications requisitioned during the reporting period
  2. Adding a new section to indicate the total amount paid for all Cadastre Applications (i.e. sum of fees paid for Record Plan services)
  3. Adding a further breakdown of the Lease related services based on the property estate type (i.e. Customary, Freehold or Government)

138

Map requires Measure Tool

Added new map tool (Measure) so users with the Measure Tool role can measure distances on the map.

139

Password Encryption fails to correctly encrypt some words

Fixed password encryption algorithm so any word can be used for a password.

145

Lot 338 Plan 4079

Updated the CFC to exclude survey services from the Unregistered dealings list.
