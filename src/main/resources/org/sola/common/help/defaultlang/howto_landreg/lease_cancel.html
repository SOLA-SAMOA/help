Cancel Lease

To surrender or terminate a registered lease or sublease, use the Termination of Lease or Sub Lease service.

Steps

    1. Start the Termination of Lease or Sub Lease service. The Property Details screen will open.
    2. On the Rights / Restrictions tab, select the Lease right you intend to cancel. If the Lease right is current and there are no pending changes for the Lease , theExtinguish tool will enable. Click Extinguish to open the Lease screen. 
    3. Update the lease details as appropriate and set the Memorial text. If the lease was migrated from LRS, you may need to populate the right holder and document details before you can save your changes.
    4. ClickCreate & Save and close the Lease screen.
    5. Canceling a right creates a new pending version of that right. If you need to make further changes to the lease information, select the pending Lease in the Rights / Restrictions tab and click theEdit tool.
    6. If you need to reverse the cancellation, use theRemove tool on the Rights / Restrictions tab or remove the pending right.
    7. Save changes to Property Details andComplete the service
