Record Miscellaneous

From time to time you may need to register a miscellaneous memorial on the property. You can add that memorial using the Record Memorial service.

Steps

    1.  Start the Record Memorial service. The Property Details screen will open.
    2. On the Rights / Restrictions tab the Miscellaneous right will be selected as the default Right type. If not, select it from the dropdown list. Click theCreate tool to create a pending Miscellaneous right and open the Miscellaneous screen .
    3. In the Miscellaneous screen enter the details for the memorial as required.
    4. ClickCreate & Save and close the Miscellaneous screen.
    5. If you need to make further changes to the miscellaneous information, select the pending Miscellaneous right in the Rights / Restrictions tab and click theEdit tool.
    6. If you need to remove the pending Miscellaneous right details completely, use theRemove tool on the Rights / Restrictions tab.
    7. Save changes to Property Details andComplete the service.
