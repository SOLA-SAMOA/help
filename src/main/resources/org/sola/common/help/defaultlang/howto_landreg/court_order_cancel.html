Cancel Court Order

To cancel a court order over a property, use the Remove Court Order service.

Steps

    1. Start the Remove Court Order service. The Property Details screen will open.
    2. On the Rights / Restrictions tab, select the Court Order right you intend to cancel. If the Court Order right is current and there are no pending changes for the Court Order, theExtinguish tool will enable. Click Extinguish to open the Court Order screen. 
    3. Update the court order details as appropriate and set the Memorial text. If the court order was migrated from LRS, you may need to add document details before you can save your changes.
    4. ClickCreate & Save and close the Court Order screen.
    5. Canceling a right creates a new pending version of that right. If you need to make further changes to the court order information, select the pending Court Order in the Rights / Restrictions tab and click theEdit tool.
    6. If you need to reverse the cancellation, use theRemove tool on the Rights / Restrictions tab or remove the pending right.
    7. Save changes to Property Details andComplete the service
