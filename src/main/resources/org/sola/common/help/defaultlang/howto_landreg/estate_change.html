Change the Type of Estate

The estate type for the property is determined by its primary right and can be Freehold, Leasehold, Government or Leasehold. The Change Estate Type service can be used to change the primary right of a property. For example, changing the title from Customary ownership into private Freehold ownership, Customary to Government ownership or Freehold to Government ownership.

There may be situations where the property information migrated from LRS does not include ownership details. In this case you can add the ownership details to the property using the Change Estate Type service.

The Change Estate Type service will not allow you to cancel the existing primary right on the property. If you must change the primary right for a property you will also need to use the Correct Registry (Cancel Right) service to cancel the existing primary right.

Note that changing the estate type of a property is only required when the property must maintain its existing folio reference. Where new parcels have been created as a result of a survey or complied plan, then the existing property record should be cancelled (Cancel Title service) and new titles created (Create New Title service). 

Steps

    1. Check the property to determine if it has an existing primary right that must be cancelled. If so, ensure the Correct Registry (Cancel Right) service is added to the application.
    2. Start the Change Estate Type service. The Property Details screen will open.
    3. On the Rights / Restrictions tab, select the new primary right for the property and click theCreate tool to create a new pending primary right and open the Ownership screen. 
    4. Fill in the ownership, share and party details as appropriate and set the Memorial text.
    5. ClickCreate & Save and close the Ownership screen.
    6. If you need to make further changes to the ownership information, select the pending primary right in the Rights / Restrictions tab and click theEdit tool.
    7. If you need to remove the pending ownership details completely, use theRemove tool on the Rights / Restrictions tab.
    8. Save changes to Property Details andComplete the service.
    9. If the property has an existing primary right, use the Correct Registry (Cancel Right) service to Extinguish the original primary right.
