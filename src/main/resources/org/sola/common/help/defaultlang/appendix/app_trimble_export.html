Appendix 2 - Trimble Pathfinder Software Configuration for SOLA Export CSV File

SOLA CSV Format definition:

    • First column: Is a unique identifier of the point within the file. It can start with 1 and incrementing by one.
    • Second column: X coordinate.(Easting)
    • Third column: Y coordinate. (Northing)

The values are separated by commas

Example:

Id,X,Y

   

1,570559.75,5931649.69

In this example the coordinate system  will be  UTM,  Zone 31N and the datum is WGS84. FOR SAMOA – Ensure the Samoa Map Grid coordinate system is configured at step 7

Steps

    1. Start Pathfinder Office and create/select a project.
    2. Select Utilities > Export and New:

 

New Export Setup

    1. In the New Setup Window enter a Setup name (for example:  _Sola CSV ). Note: The underscore will bring the SOLA format to the top of the list later on.
    2. Select New Setup and scroll for Configurable ASCII and tap OK.

Coordinate System Setup

    1. The Export Setup Properties Window will open. Select the Coordinate System Tap.
    2. Select Use Export Coordinate System and tap Change… The Export Window will open.
    3. Select System: UTM
    • Zone: Select the Zone for Samoa Map Grid (SMG)
    • Datum: Select the Datum for Samoa Map Grid (SMG)
    1. Click OK. The Export Setup Properties Window will open again.

Export Template Name

    1. Select the Configurable ASCII Tab and New…
    2. Enter a name for your new format (Example SOLA) and click OK. The ASCII Export Template Editor page will open

Export Template Editor

    1. Enter the Output File Extension : CSV
    2. Apply to : Points only
    3. Select the Field Format : Delimited
    4. Select the Delimiters Field : Comma (,)
    5. Select the following output definitions from the Macro Palette with left mouse click:
    • {Feature ID}
    • {Easting}
    • {Northing}

Note: Easting refers to the eastward-measured distance (or the x-coordinate) and Northing refers to the northward-measured distance (or the y-coordinate).

    1. To use the define Export system select Select Utilities > Export, select your data file and Choose an Export Setup:

Choose Export Setup
