Add a Traverse Mark or District Boundary

To add a Traverse Mark or a District Boundary, us the Add New Traverse Mark or the Add New District Boundary tools during either Record Plan or the Change Map Services. You can also move or edit the new features using theSelect Spatial Feature tool and/or theEdit Spatial Feature tool.

Be aware that although you can see the changes to the shape of the feature, they will not be reflected on the map for others to see until the application is approved.
