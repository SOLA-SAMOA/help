Make an Historic Deed Current

All of the deeds migrated into SOLA were migrated with an historic/cancelled status to prevent further actions or changes to those deeds. If it becomes necessary to lodge a dealing against an historic deed, the deed must be updated to a Current status to allow the dealing to be lodged. This can be achieved using the  Make Current feature that is available on the Property Details screen for historic deeds

Important: Only users with the Make Property Current security privilege can use the Make Current functionality. This privilege is part of the Team Leader security role.

Steps

    1. Use Property Search to locate the property to print the CoT and open Property Details.
    2. If you have the Make Property Current security privilege, you will see the Make Current button appear next to the Terminate button. Click Make Current
    3. At the prompt, confirm you have the correct deed and click Yes.
    4. SOLA will update the deed and change its status to Current. The deed will then allow dealings to be submitted against it.
