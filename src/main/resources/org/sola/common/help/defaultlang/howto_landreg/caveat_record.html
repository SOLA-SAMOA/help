Record Caveat

Caveats are used to block any changes on the property until such time as the caveat is cancelled. This means it is not possible to register, change or cancel any rights or restrictions on the property while the caveat is in force. SOLA implements this restriction using business rules that prevent services other than Change Caveat and Removal of Caveat from being completed while a pending or current caveat is associated with the property.

Use the Record Caveat service to register a caveat over a property.

Steps

    1.  Start the Record Caveat service. The Property Details screen will open.
    2. On the Rights / Restrictions tab the Caveat restriction will be selected as the default Right type. If not, select it from the dropdown list. Click theCreate tool to create a pending Caveat restriction and open the Caveat screen.
    3. In the Caveat screen enter the right holder details for the caveat . This is the party or parties that will hold the caveat.
    4. Fill in the Registration date and update the Memorial text.
    5. ClickCreate & Save and close the Caveat screen.
    6. If you need to make further changes to the caveat information, select the pending Caveat in the Rights / Restrictions tab and click theEdit tool.
    7. If you need to remove the pending Caveat details completely, use theRemove tool on the Rights / Restrictions tab.
    8. Save changes to Property Details andComplete the service.
