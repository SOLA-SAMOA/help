Release 1403a

This section summarises the changes made to the SOLA Samoa Desktop application or it’s configuration for Release 1403a. Further details for each change can be found in the SOLA Samoa Lighthouse Project (https://flossola.lighthouseapp.com/projects/92838-samoa/overview) using the Lighthouse Ticket # as reference.

Ticket #

Title

Summary

57

Power of Attorney

User was able to link historic Power of Attorney documents new applications.

94

Total Check Shares business rule has false failure

Business rule was incorrectly calculating the total of all shares if the shares had a large denominator.  

97

Unable to create mortgage on 2891/5631

Fix for LRS migration issue where some notations were null. 

98

Unable to save manually entered survey coordinates

Survey Point Id was not being allocated correctly.

99

1/12 Shares

Database connection issue that was preventing applications from being approved if there were a large number of ownership shares.

102

Unable to locate existing lot when searching from Property Parcel Search

Updated search so user can enter \s to indicate the location of a space in the parcel name improving the accuracy of the search results.

103

Error when attempting to cancel a title and create a new title over the same parcel

Steps provided so user could correct data issues with application 40451.

104

Report the number of times a CFC has been printed

Added Certificate Print Log dialog to capture summary details whenever a property report is printed. Print details are displayed on the Property Details form.

106

Lodgement Report does not count services that are cancelled on Requisitioned Applications

Updated report to correctly count the number of cancelled applications.

107

Prompt User to change password on a regular basis

Password expiry implemented. Users are prompted to change their passwords at regular intervals (default interval is 90 days).

108

Historic tables for user groups and roles

Added history tables for the group and role tables so that any changes to these tables are tracked.

110

Unable to process applications on lots with a large number of owners due to issues with Distributed Transactions.

As with Ticket 99, a database connection issue was preventing the application from being approved. Reconfiguration of Glassfish fixed the issue.

111

Parcel block number becomes plan number when creating a new property

Removed block number from the title reference.

115

Tool to add, edit and remove villages

Added new map tools allowing users to add, edit and remove village points from the map.

116

Remove and/or Merge Road Polygons

Added Merge feature to Spatial Unit Edit dialog to allow polygon spatial units (i.e. Roads and Hydro parcels) to be merged together to create a new polygon.

125

Update webstart to work with the latest Java updates

Several updates to the SOLA Samoa Desktop deployment package to make it compatible with the latest Java security requirements.

126

User Search screen does not resize correctly.

Screen resizing issues fixed.

127

General improvements from SOLA FAO

Five functional improvements in SOLA FAO were migrated into SOLA Samoa. These included;

  1. Support for network folders (allows SOLA to connect to file shares on other computers)
  2. Improvements to Image Preview so that the thumbnail is cached by the client
  3. Support for resizing the Image Preview in the File Attachment dialog
  4. Recording the User preference for the folder to attach documents from
  5. Allow filtering of local files when attaching them to make it easier to find specific files in folders with 100's or 1000's of documents.

128

Easements to display in the Land Description on CFC

Updated CFC so that any easements registered on a property are listed in the Land Description section of the CFC instead of the Second Schedule.

129

Proclamation changes Estate Type

Updated Record Proclamation service to allow the user to change the estate type of a property rather than create a Proclamation right.

130

Service corrections

Updated the display names for several services.

132

Need to record Land and Titles Court Judgments in SOLA

Added the Lands and Title Courts Judgment as a document type so that judgments can be scanned and loaded into SOLA.

133

Critical Rule error when attempting to subdivide disjoint parcels

Updated business rules so that SOLA can support subdivision of parcels that may be separated by a road or other feature.

134

SOLA Map Viewer does not display when running web start

Fixed issue with source code references to allow map viewer to display correctly.
