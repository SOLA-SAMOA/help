Record Plan

The Record Plan screen allows you to subdivide/split or merge/amalgamate parcels. It can be used to capture new parcel information obtained from a plan (e.g. Survey, Complied, Registry, etc). It can be accessed bystarting the Record Plan service from the Application Details Service tab.

Record Plan with Target Parcel selected

The Record Plan screen extends the SOLA Map Viewer with additional tools to aid spatial editing of parcels as well as tools to add or edit existing spatial features (i.e. hydro parcels and road centerlines). The additional tools provided are

Create / Link Survey Points

Create New Parcel

Select a Boundary to Change
