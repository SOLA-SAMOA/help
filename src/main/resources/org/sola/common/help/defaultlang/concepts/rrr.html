Rights, Restrictions and Responsibilities

RRR are the rights, restrictions and responsibilities that can be registered on a property. The primary right is used to indicate the main estate type for the property (i.e. Customary, Freehold, Government or Leasehold). Every property should have one primary right although properties migrated from LRS may omit the primary right if one could not be determined during the migration. The services that can be used to create, change or cancel each type of RRR are also indicated in the table. Note that the Correct Registry service can be used to create or change any RRR while the Correct Registry (Cancel Right) service can be used to cancel any RRR.
