Record Transmission

To record a new transmission on a property, use the Record Transmission service.

Steps

    1. Start the Record Transmission service. The Property Details screen will open.
    2. On the Rights / Restrictions tab the Transmission right will be selected as the default Right type. If not, select it from the dropdown list. Click theCreate tool to create a pending Transmission right and open the Transmission screen .
    3. In the Transmission screen enter the details as required.
    4. ClickCreate & Save and close the Transmission screen.
    5. If you need to make further changes to the transmission information, select the pending Transmission right in the Rights / Restrictions tab and click theEdit tool.
    6. If you need to remove the pending Transmission details completely, use theRemove tool on the Rights / Restrictions tab.
    7. Save changes to Property Details andComplete the service.
