Correct Registry (Cancel Right)

Use the Correct Registry (Cancel Right) service to cancel existing rights on the property. This service will also allow cancellation of the primary right when changing the property estate type.

Steps

    1. Start the Correct Registry (Cancel Right) service. The Property Details screen will open.
    2. Use the Rights / Restrictions tab,Extinguish rights or restrictions as appropriate. 
    3. Save changes to Property Details andComplete the service.
