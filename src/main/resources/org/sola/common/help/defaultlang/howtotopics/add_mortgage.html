Add Mortgage Details to a Property

You can add mortgage details to a property using either the Register Mortgage service or the New Digital Title service. Using an assigned application, start the mortgage related service from the Application Details Services tab to open the Property Details screen. On the Rights / Restrictions tab select the Mortgage right (this may be pre-selected for the Register Mortgage service) and click Create. This will open the Mortgage Details screen



Enter the mortgage details and click Create. The Property Details screen will display with a pending Mortgage right. Click Save to save the mortgage details. To complete the change in mortgage details, complete the mortgage related service using the Complete tool on the Application Details Services tab and then approve the application.
