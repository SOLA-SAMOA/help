Change Body Corporate Rules or Address for Service

The Body Corporate Rules and Address for Service are listed as rights on the Common Property folio. Both rights can be changed using the Change Body Corporate service.

Steps

    1. Lodge an application with the Change Body Corporate service. You must reference the Common Property folio for the Unit Title Development on the Property tab of Application Details (e.g. CP/11000).
    2. Ensure you are Assignee for the application with the Change Body Corporate service and open it from your Dashboard.
    3. Start the Change Body Corporate service. The Property Details screen will open.
    4. On the Rights / Restrictions tab, select the Body Corporate Rules or Address for Service right you intend to change. If the right is current, theVary tool will enable. ClickVary to open the Body Corporate Rules or Address for Service screen. 
    5. Update the right details as appropriate and set the Memorial text.
    6. ClickCreate & Save and close the screen.
    7. Varying a right creates a new pending version of that right. If you need to make further changes to the right information, select the pending right in the Rights / Restrictions tab and click theEdit tool.
    8. If you need to remove the pending right details completely, use theRemove tool on the Rights / Restrictions tab.
    9. Save changes to Property Details andComplete the service.
    10. Continue processing the application and use theApprove tool to register changes to the Body Corporate information.
